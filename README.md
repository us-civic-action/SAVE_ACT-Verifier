## üéØ Purpose

This tool simplifies the complex requirements of the **Safeguard American Voter Eligibility (SAVE) Act**, providing a clear, step-by-step verification process for users across all 50 states. It features:

- **State-Specific Logic**: tailored questions based on local residency and ID laws.
- **Privacy-First Architecture**: runs entirely in the browser; no personal data is stored or transmitted.
- **Accessible Design**: high-contrast modes, large text support, and screen reader optimization.

## üöÄ Quick Start

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/us-civic-action/SAVE_ACT-Verifier.git
    cd SAVE_ACT-Verifier
    ```

2. Install dependencies:

    ```bash
    npm install
    # or
    yarn install
    ```

3. Start the development server:

    ```bash
    npm run dev
    ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.
